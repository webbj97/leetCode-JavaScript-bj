## 二、算法题
### 题目
**示例:**
```
输入: 1->2->3->4->5->NULL
输出: 5->4->3->2->1->NULL
```
进阶:
你可以迭代或递归地反转链表。你能否用两种方法解决这道题？

## 解法一 （新建节点）

#### 思路
    * 创建一个空节点
    * 保存当前节点的后置节点,再将之后递归到下一个节点
#### 代码

```js
var reverseList = function(head) {
    let cur = head  //  当前节点
    let prev = null //  前置节点初始化（head节点无前置节点）
    while(cur){
        let temp = cur.next //  保存当前节点的后置节点 之后递归到下一个节点
        cur.next = prev
        prev = cur
        cur = temp
    };
    return prev
};
```
