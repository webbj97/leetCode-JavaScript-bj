58.最后一个单词的长度

最后一个单词的长度

给定一个仅包含大小写字母和空格 ' ' 的字符串，返回其最后一个单词的长度。

如果不存在最后一个单词，请返回 0 。

说明：一个单词是指由字母组成，但不包含任何空格的字符串。

示例：

输入 输出
输入: "Hello World" 输出: 5

思路一（split）

考虑直接以空格为界限转换为字符串数组，但是特殊的结构需要注意

"str " --> [str,'','','']，数组的长度为4，但是他们是应该被忽略的

var lengthOfLastWord = function(str) {
    if(!str){
        return 0
    }
    str = str.split(" ")
    var len = str.length-1
    for(var i = len; i >= 0; i--){
        if(str[i]){
            return str[i].length
        }
    }
    return 0
};
