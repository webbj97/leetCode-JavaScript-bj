203.移除列表元素

移出列表元素

删除链表中等于给定值 val 的所有节点。

示例:

输入: 1->2->6->3->4->5->6, val = 6 输出: 1->2->3->4->5

解法一（常规）

var removeElements = function(head, val) {
    if (head == null) return null;
    var temp = {
        val: -1,
        next: head
    }
    var search = temp;
    while (search.next) {
        if (search.next.val == val) {
            search.next = search.next.next;
        } else {
            search = search.next;
        }
    }
    return temp.next;
};
