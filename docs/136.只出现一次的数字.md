## 解法一（数学方法）

![在这里插入图片描述](https://img-blog.csdnimg.cn/20191230182929587.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2piajY1Njg4Mzl6,size_16,color_FFFFFF,t_70)
```js
var singleNumber = function(nums) {
  var res = 0;
  if(nums.length < 2){
    return nums[0]
  }
  for(var i = 0 ;i < nums.length; i++ ){
    res ^= nums[i] //亦或  运用异或运算，任何一个数字异或它自己都等于0。
  }
  return res
};
```
