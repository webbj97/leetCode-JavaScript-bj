## 给定一个未排序的整数数组，找出其中没有出现的最小的正整数。
| 输入| 输出  |
|--|--|
|  [1,2,0 ] | 3 |
|  [3,4,-1,1] | 2 |
| [7,8,9,11,12] | 1 |

### 解法一（常规思路）
![在这里插入图片描述](https://img-blog.csdnimg.cn/20191008141251466.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2piajY1Njg4Mzl6,size_16,color_FFFFFF,t_70)
```js
var firstMissingPositive = function(nums) {
    var arr = [],i = 1
    nums.forEach(e => {
        if(e > 0){
            arr.push(e)
        }
    });//留下正数
    while(arr.indexOf(i) >= 0 ){
        i++
    }
    return i
};
```
