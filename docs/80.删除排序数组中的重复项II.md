<h1 align-center>删除排序数组中的重复项</h1>

## 解法一（辅助下标）
```js
var removeDuplicates = function(nums) {
    let l = nums.length
    let par = 0
    for (let i = l - 1; i > 0; i--) {
        if (nums[i] != nums[i - 1]) {
            par = 0
        } else {
            par++
            if (par > 1) {
                nums.splice(i, 1)
            }
        }
    }
    return nums.length
};
```

## 解法二
利用保留2个题干，逐个比较，但比较间隔为2（已经排好序的数组）
```js
var removeDuplicates = function(nums) {
    let l = nums.length
    for (let i = 2; i < l; i++) {
        if (nums[i] == nums[i - 2]) {
            nums.splice(i, 1)
            i--
        }
    }
    return nums.length
};
```
