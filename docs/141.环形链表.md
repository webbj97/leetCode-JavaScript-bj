<h1 align=center>环形链表</h1>


给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。



## 解法一
![在这里插入图片描述](https://img-blog.csdnimg.cn/20191230184648891.jpg?x-oss-process=image/watermark,type_ZmFuZ3poZW5naGVpdGk,shadow_10,text_aHR0cHM6Ly9ibG9nLmNzZG4ubmV0L2piajY1Njg4Mzl6,size_16,color_FFFFFF,t_70)
```js
var hasCycle = function(head) {
    var setBox = new Set()
    while(head != null){
        if(setBox.has(head.next)){
            return true
        }else{
            setBox.add(head.next)
            head = head.next
        }
    }
    return false
};
```
