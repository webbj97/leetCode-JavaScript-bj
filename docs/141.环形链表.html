141.环形链表

环形链表

给定一个链表，判断链表中是否有环。

为了表示给定链表中的环，我们使用整数 pos 来表示链表尾连接到链表中的位置（索引从 0 开始）。 如果 pos 是 -1，则在该链表中没有环。

解法一

var hasCycle = function(head) {
    var setBox = new Set()
    while(head != null){
        if(setBox.has(head.next)){
            return true
        }else{
            setBox.add(head.next)
            head = head.next
        }
    }
    return false
};
